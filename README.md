五張撲克牌遊戲:玩家原先有10000總資金，可自訂下注金額，透過「發牌」與「換牌」機制來獲得最終手牌。程式會自動判斷手牌牌型（如同花順、三條、兩對等），並根據對應的賠率發放獎金回總籌碼中。

遊戲流程

1.初始資金：玩家初始擁有 10,000 點籌碼。
2.下注金額：在「押注金額」欄位輸入數字後，點擊 [下注] 按鈕。
3.第一輪發牌：點擊 [發牌]，系統會隨機洗牌並發出 5 張牌。
4.選擇留牌/換牌：點擊桌面上的撲克牌，被點擊變為「牌背」狀態的牌代表將在下一輪被替換。
5.確認換牌：點擊 [換牌]，系統會替換掉蓋住的牌並顯示最終結果。
6.結算結果：點擊 [判斷牌型]，程式將判定結果並自動計算獎金，更新籌碼後請重新下注開始新的一局。

鍵盤測試模式 (發牌後使用)

若要測試特定高階牌型，可在發牌後按下：Q:同花大順 W:同花順 E:同花 R:鐵支 T:葫蘆 Y:三條
